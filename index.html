Welcome!

This simple example will show you how to send emails with an HTML form and PHP!

I've included some comments in sendMail.php to explain how to add variables. If confused, just make a new label tag with a name that is read by PHP in $_POST['variablename'];. This'll take the posted data and allow it to be usable by PHP. If you encounter errors with this, Google will be your friend. Or drop a bug report on my github!
